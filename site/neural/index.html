02Invasive vs non-invasive

03The edge pipeline

Surface sensors → on-device DSP (the console above) → the BRMSTE GI bridge. No cloud round-trip for the signal path.

SENSE

Non-invasive surface sensors

EEG (scalp), EOG (eye), EMG (muscle), PPG (pulse). Consumer-grade, dry-electrode, Web Serial / Web Bluetooth.

PROCESS

Edge DSP

Windowing → FFT → band power (δ θ α β γ) → ratio indices & state inference. Runs locally — the exact code above.

BRIDGE

GI → GSI → QGSI

Intent & state stream into BRMSTE General Intelligence. The human stays in the human lane; the edge signs, judgment signs.

  • Static and backend-free — serve the folder (e.g. python3 -m http.server) and it runs entirely client-side; no signal data leaves the browser.
  • The DSP is verified by a pure-Node test suite (dsp.test.mjs) and an in-browser self-test (badge above).
  • The synthetic source is labelled synthetic. A real device runs the identical pipeline.
