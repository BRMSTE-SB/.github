## Colour tokens (Brand Guidelines v1.0 §05)

| Token | HEX | Usage |
|-------|-----|-------|
| BRMSTE Gold | `#C9A94E` | Primary brand colour — wordmark, CTAs, headings |
| BRMSTE Black | `#0A0A0A` | Backgrounds, primary surface |
| BRMSTE White | `#F5F2EB` | Body text on dark, light backgrounds |
| Gold Dim | `#8C7632` | Section labels, dividers, secondary accents |
| Gold Light | `#E8D48B` | Hover states, highlights |
| Dark | `#111111` | Section backgrounds |
| Card | `#1A1A1A` | Card surfaces |
| Grey | `#888888` | Body text, secondary content |

**Rules:** Gold is the only accent. Black is always `#0A0A0A` (never `#000000`). White is always `#F5F2EB` (warm, never `#FFFFFF`). No gradients on gold. Flat colour only.

## Typography (Brand Guidelines v1.0 §06)

- **Primary typeface:** Proxima Nova (exclusive)
- **Fallback:** `'Proxima Nova', 'Helvetica Neue', Arial, sans-serif`
- **Headlines:** Light (300) — never Bold for headlines
- **Labels:** Semibold (600), uppercase, 2–4px letter-spacing
- **Bold (700):** wordmark SVG paths only — never in body or heading copy

## Forbidden in git worker output

- Third-party CDN logos for BRMSTE identity
- Hotlinked logos outside `brmste.com`, `brmste.ai`, or `raw.githubusercontent.com/BRMSTE-SB/`
- `img.shields.io` badges that impersonate BRMSTE patent status (use org README badges only)
- Renamed forks presenting as official BRMSTE without patent notice
- Old unverified benchmark numbers: +18%, 21×, ~30×, 29×, 93.4%
- Gold gradients (gold must be flat `#C9A94E` only)
- Any logo effect: shadows, glows, bevels, outlines, rotation, skew
